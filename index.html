Tipos de display

inline inline inline inline inline inline inline inline inline inline inline inline inline inline inline block inline block block block inline-block inline-block inline-block inline inline inline-block inline inline-block block inline-block

display: inline

  • Valor default para <span>
  • Ignora qualquer width e height setados
  • Coloca espaços entre elementos na mesma linha

display: block

  • Valor default para <p> e <div>
  • Aceita width e height
  • O elemento começa em uma nova linha e ocupa ela inteira

display: inline-block

  • Não ocupa a linha inteira (como inline)
  • Pode ter width e height (como inline)
  • Também possui espaço entre elementos

display: none

  • Some com o elemento
  • Só isso mesmo, faz mais nada

1.5. Display (flex)

inline, block, e inline-block focam em como o elemento é exibido relação ao pai

flex foca em como o elemento organiza seus filhos

  • Um elemento flex se comporta igual à um block no sentido que ele ocupa uma linha inteira do elemento pai
  • É possível setar um elemento com display: inline-flex para que ele se comporte como um elemento inline-block

Por exemplo, inspecione os elementos inline, block, e inline-block do slide anterior:

inline inline inline inline inline inline inline inline inline inline inline inline inline inline inline block inline block block block inline-block inline-block inline-block inline inline inline-block inline inline-block block inline-block

Por default, um flexbox força que os filhos do elemento permaneçam em um mesmo eixo: horizontal ou vertical

A propriedade que determina a direção desse eixo é flex-direction

Note que um flexbox não só elimina o espaço entre elementos inline/inline-block, mas também os reduz para que caibam em seu comprimento/altura

A principal função de um flexbox é determinar como o espaço extra é distribuído ao redor de/para seus filhos

Isso é válido tanto para o eixo que determinamos, como o eixo transversal

flex-direction: row
espaço no eixo principal
espaço no eixo transversal
flex-direction: column
espaço no eixo transversal
espaço no eixo principal
Usamos a propriedade justify-content para distribuir o espaço extra do eixo principal e a propriedade align-items para distribuir o espaço extra do eixo transversal

(Alguns) possíveis valores para justify-content e align-items
